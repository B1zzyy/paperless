# Paperless iOS App

This folder contains a native SwiftUI iOS starter app inspired by the current web app style:

- warm neutral background
- green primary accent
- rounded cards and compact spacing
- iOS-like tab bar and receipt list layout
- light haptic feedback on navigation and success actions

## Open and Run

1. Open `ios-app/PaperlessIOS.xcodeproj` in Xcode.
2. Select the `PaperlessIOS` scheme.
3. Pick an iPhone simulator (for example, iPhone 16).
4. Press Run.
